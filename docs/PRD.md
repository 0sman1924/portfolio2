# Portfolio Product Requirements Document

## Vision
Build a clean, modern, static portfolio for a **Security Analyst | DevOps Engineer** deployed on GitHub Pages.

The portfolio showcases:
- Security Analysis Reports (SOC, DFIR, Detection Engineering)
- DevOps & Software Projects
- Technical Articles & Tutorials
- Professional profile
- Resume

Current focus:
- Security Analysis
- DevOps
- SOC
- DFIR

Future focus:
- Malware Analysis

## Goals
- Professional online presence
- Demonstrate practical skills
- Easy maintenance
- Fast loading
- Mobile-first
- No backend

## Target Audience
- Recruiters
- Hiring Managers
- Security Teams
- DevOps Teams

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- MDX/Markdown
- GitHub Pages
- GitHub Actions

## Design Philosophy
- Minimal
- Technical
- Content-first
- Responsive
- Accessible
- Scalable
